# English_trainer
💻This is my first Python project - a simpe English word trainer. It helps you practise some English vocabulary (B1-B2 level) using a quiz format. The trainer is endless: words are chosen randomly, and you can keep practicing as long as you want. To stop the program, type "exit".

Features:
- random word selection
- instant answer checking
- endless practising mode

Over time, new words will be added.
